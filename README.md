## How to use

### Install via pip
```sh
pip install quilt-lang
```

### Importing
```py
import quilt_lang as _
```

### Example Usage
```py
_.pingtest()
```

For more commands, visit [this website](https://quilt-lang.richie-bendall.ml/commands/quilt_lang.html).

## How you can help

At the moment, you can help by
